Two Sum

The prompt for this problem is to find and return the first pair of two numbers in an array that add up to the target number.

Here is the code:

for (let i = 0; i < nums.length; i++) {
for (let j = i + 1; j < l; j++) {
if (nums[i] + nums[j] === target) {
return [nums[i] + " and " + nums[j]];
}
}
}

I still need to work on finding out how to make code previews work...

My approach involved two layers of iteration with a for loop. The first layer, indicated by the letter i in the loop, executes a for loop at each iteration. The second layer, indicated by the letter j, executes a check on whether those two numbers add up to the target.

This approach has a bug where it counts a space as 0.

TLDR; Entering a list of numbers separated by commas should give you the first two numbers that add up to your target.
